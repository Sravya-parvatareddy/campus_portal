# Academic Campus Event & Registration Portal

## Project Description
This project is a web-based application developed using PHP and MySQL. 
It allows students to view campus events and register for events online.
The admin can add events and view student registrations.

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP
- Apache Server

## Modules
- Admin Module
- Student Module
- Event Management Module
- Registration Module
- Database Module

## How to Run
1. Install XAMPP
2. Start Apache and MySQL
3. Copy project folder to htdocs
4. Open http://localhost/campus_event
5. Database name: campus_event (Import SQL file)
## Database Setup
1. Open phpMyAdmin
2. Create database named campus_event
3. Import campus_event.sql file
4. Run the project in XAMPP server
